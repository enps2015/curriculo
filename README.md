Currículo – Eric Pimentel
=========================

Este repositório centraliza as principais informações sobre minha carreira e formação na área de Dados. Aqui você encontrará:

*   **[cv.md](https://github.com/enps2015/curriculo/blob/main/cv.md)** — versão completa do currículo em Markdown, com histórico profissional, educação, competências técnicas, projetos e conquistas.
    
*   **[CV\_Eric\_Pimentel\_Dados.pdf](https://github.com/enps2015/curriculo/blob/main/CV_Eric_Pimentel_Dados.pdf)** — versão em PDF do currículo, preservando o layout e detalhes completos.

Como usar
---------

*   Visualize o currículo em Markdown acessando **[cv.md](https://github.com/enps2015/curriculo/blob/main/cv.md)**.
    
*   Baixe a versão em [PDF](https://github.com/enps2015/curriculo/blob/main/CV_Eric_Pimentel_Dados.pdf) se preferir compartilhar o documento.
    
*   Consulte **[courses.md](https://github.com/enps2015/curriculo/blob/main/courses.md)** para um panorama das principais certificações.
    

Por que este repositório existe?
--------------------------------

Concentrar o material em um único repositório facilita a navegação de recrutadores e colegas que desejam compreender minha trajetória e avaliar habilidades. A divisão em arquivos facilita a manutenção e atualizações futuras.

*   Este repositório será atualizado periodicamente com novos cursos, certificações e projetos relevantes.
